Harbor's desktop app is built with [Tauri v2](https://v2.tauri.app/) and gives you a full graphical interface for everything the CLI can do.

<Frame>
  <img src="/images/screenshot-light.png" alt="Harbor Desktop App" />
</Frame>

## Pages

The app is organized into four main sections, accessible from the sidebar:

<CardGroup cols={2}>
  <Card title="Fleet" icon="ship" href="/desktop/servers">
    Manage your docked MCP servers — add, remove, enable, or disable them.
  </Card>
  <Card title="Ports" icon="plug" href="/desktop/hosts">
    Connect to hosts like Claude Code, VS Code, and Cursor, then sync your fleet.
  </Card>
  <Card title="Scout" icon="magnifying-glass" href="/desktop/marketplace">
    Search and install MCP servers from the official MCP Registry.
  </Card>
  <Card title="Helm" icon="gear" href="/desktop/settings">
    Configure the gateway, manage secrets, adjust themes, and check for updates.
  </Card>
</CardGroup>

## System Requirements

- **macOS**: Apple Silicon or Intel (macOS 11+)
- **Windows**: x86_64
- **Linux**: x86_64
