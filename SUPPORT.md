# Support

Thanks for using Harbor! Here's how to get help.

## Documentation

The best place to start is the [Harbor Documentation](https://docs.harbormcp.ai), which covers:

- [Installation](https://docs.harbormcp.ai/quickstart/installation)
- [Your First Server](https://docs.harbormcp.ai/quickstart/first-server)
- [CLI Commands](https://docs.harbormcp.ai/cli/commands)
- [Vault & Secrets](https://docs.harbormcp.ai/guides/vault)
- [Team Fleet Sync](https://docs.harbormcp.ai/guides/team-sync)

## Questions & Discussions

For general questions, ideas, or community discussion:

- **[GitHub Discussions](https://github.com/JoshuaShunk/Harbor/discussions)** — Ask questions, share configs, propose features

Please search existing discussions before opening a new one.

## Bug Reports & Feature Requests

Found a bug or have a feature idea?

- **[Bug Report](https://github.com/JoshuaShunk/Harbor/issues/new?template=bug_report.md)** — Report issues with detailed reproduction steps
- **[Feature Request](https://github.com/JoshuaShunk/Harbor/issues/new?template=feature_request.md)** — Suggest new functionality

Please check [existing issues](https://github.com/JoshuaShunk/Harbor/issues) before filing a new one.

## Security Issues

If you discover a security vulnerability, **do not open a public issue**. Please see our [Security Policy](SECURITY.md) for responsible disclosure instructions.

## Response Times

Harbor is maintained by a small team. We aim to:

- Acknowledge new issues within a few days
- Respond to security reports within 48 hours
- Review pull requests weekly

For urgent issues, please clearly mark them in the title.

## Contributing

Want to contribute? See our [Contributing Guide](CONTRIBUTING.md) for how to get started with development.
